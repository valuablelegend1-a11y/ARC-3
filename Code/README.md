**Python Libraries:**
All libraries used are part of the Python Standard Library; no external pip installations are required.
* tkinter/simpledialog: GUI framework and processing
* socket/threading: Networking and concurrent data handling.
* json, math, time, os, csv, datetime: Data processing and system management.

**Arduino Libraries:**
These are the libraires used in the Arduino firmware, they can be installed via the Arduino Library Manager (Ctrl+Shift+I).
* Wire.h: Handles I2C communication.
* WiFiS3.h: Standard library for Arduino UNO R4 WiFi.
* Adafruit PWM Servo Driver Library: Created by Adafruit. Used to control the PCA9685 PWM driver.
