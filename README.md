# ARC-3
My ARC-3 is a 3 DoF and 3 fingered robotic arm, focusing on adaptability and integrating UDP capabilities for use away from a computer, and later adding interchangeable and modular bases for simple relocation. It will have an orange and black color scheme and designed to be sleek and functional. It will be created using an Arduino R4 WiFi for the controller, a 5V 8A AC to DC power adapter for power to both the servos and the arduino, a PCA9685 PWM Servo motor driver board to handle power,  MG995 servos for the waist, shoulder, and elbow joints to have high torque and power even under load, and a seperately oriented MG995 for the fingers, beacuse why not have a little extra power.
